# BFS Tree-Printer-Scala

This is scala implementation of the BFS algorithm.
 
Breadth-first search (BFS) is an algorithm for traversing or searching tree or graph data structures

INPUT
-
```
      1
     / \
    3   5
   / \   \
  2   4   7
 / \   \
9   6   8
```


OUTPUT
-
```
1
3 5
2 4 7
9 6 8 
```
